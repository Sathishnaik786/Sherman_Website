## 🎨 UI/UX Design System

The platform follows a **"Premium Dark"** design philosophy, optimized for high-density information display and executive-level engagement.

- 🧊 **Glassmorphism:** Consistent use of `backdrop-filter` and translucent borders for a sophisticated depth effect.
- 🍱 **Bento Grids:** Dense, organized layouts for services and product features.
- 🌑 **Dark Mode by Default:** Optimized for low-light environments and energy efficiency.
- ⚡ **Micro-Interactions:** Subtle haptic-like animations on buttons, cards, and navigation.

## 📂 Folder Structure

```bash
src/
├── components/
│   ├── layout/         # Navbar, Footer, MegaMenu, Overlay
│   ├── sections/       # Homepage modular sections (Hero, Bento, CTA)
│   ├── services/       # Specialized components for Service pages
│   ├── products/       # Specialized components for Product pages
│   └── ui/             # Core UI primitives (shadcn)
├── pages/
│   ├── services/       # AI-ML, E-Governance, Web-Mobility, etc.
│   ├── products/       # Land-Acquisition, Farm2Plate, TAMRA, etc.
│   ├── blog/           # Listing and detailed article pages
│   └── Home.tsx        # Enterprise landing page
├── data/               # Centralized content management (blogs, items)
├── hooks/              # Custom React logic
└── lib/                # Utility functions & Tailwind merge
```
